# LinkedIn assets

Place source and exported image/video files here.

Naming: `YYYY-MM-DD-slug.ext` to match the draft it belongs to.

Recommended sizes:
- Link / landscape image: 1200×627
- Square image: 1080×1080
- Article cover: 1920×1080

## Brand assets

Logos, favicons, fonts, and the color palette are shared in
[../../brand/assets/](../../brand/assets/). See
[../../brand/brand-voice.md](../../brand/brand-voice.md) for colors, typography,
and logo usage. Quick reference:

- **Primary blue:** `#070656` · **Coral accent (CTAs):** `#ff7065` · full palette in brand-voice.
- **Font:** Quicksand — Bold (700) headings, Medium (500) body.
- **Logo:** use the **white** logo on dark navy, **blue** on light backgrounds.

## Post visuals

Single-image, multi-image, and document/PDF visuals are authored as SVG (see the
[design-linkedin-post-visuals skill](../../.github/skills/design-linkedin-post-visuals/SKILL.md))
and rendered with [../../export-tooling/](../../export-tooling/). LinkedIn gets a
`document.pdf` for document posts plus PNGs in each post's own `export/linkedin/`
folder. Don't hand-place exported files here.
